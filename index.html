Engine rules summary
Rules implemented: Core Z-pattern scan (top→bottom, left→right) · Add: fill first fitting empty space, else append after last · Delete: remaining widgets move up, column stays (repeat until stable) · Reposition: swap when crossing center, replaced widget → first fitting open slot, settle + gravity · Resize (Quick Action size row): full re-pack in scan order, no toast.
